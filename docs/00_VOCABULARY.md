# HENRY AI CORPORATION — CANONICAL VOCABULARY

**Rule:** Every word means ONE thing. No overloading.

| Term | Definition |
|---|---|

| **Dark Factory** | Internal codename for CPA acquisition + AI transformation operation. |

## THE AGENTS

|---|---|---|
| **ORCHESTRATOR** | Routes all tasks. Manages the team. | `/ORCHESTRATOR` |
| **CEO** | Business strategy, acquisitions, decisions. | `/CEO` |
| **CFO** | Finance, valuations, SBA, deal structure. | `/CFO` |

| **OPERATIONS** | Sprint management, execution tracking. | `/OPERATIONS` |

## SUB-AGENT NAMING
```
SUB-[PARENT TITLE]-[TWO DIGIT NUMBER]
Example: CFO spawns 3 → SUB-CFO-01, SUB-CFO-02, SUB-CFO-03
```
